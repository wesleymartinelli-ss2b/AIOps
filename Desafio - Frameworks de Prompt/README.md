Material do desafio do módulo de Engenharia de Prompts. Cenários fictícios da **Hill Valley Tech**.

## Modelo utilizado

| Campo | Valor |
|-------|--------|
| **Provider** | Cursor |

## Idioma

| Tipo | Idioma |
|------|--------|
| Prompts, justificativas, documentação | PT-BR |
| Dockerfile, YAML, Terraform, SQL, bash e comentários nesses artefatos | en_US |


Cada pasta `qNN-*` contém:

| Arquivo | Conteúdo |
|---------|----------|
| `prompt.md` | Texto exato do prompt (fase 2) |
| `modelo.md` | Modelo usado + 1 linha de justificativa da escolha |
| `output.md` | Resposta do modelo (íntegra ou trecho relevante) |
| `justificativa.md` | Como os componentes do framework aparecem no prompt |

Na Q08, `justificativa.md` inclui comparação com ≥2 frameworks alternativos.
